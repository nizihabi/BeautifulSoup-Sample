当我们在传入函数时，有些时候，不需要显式地定义函数，直接传入匿名函数更方便。2>>> list(map(lambda x: x * x, [1, 2, 3, 4, 5, 6, 7, 8, 9]))
[1, 4, 9, 16, 25, 36, 49, 64, 81]
def f(x):
    return x * x
用匿名函数有个好处，因为函数没有名字，不必担心函数名冲突。此外，匿名函数也是一个函数对象，也可以把匿名函数赋值给一个变量，再利用变量来调用该函数：>>> f = lambda x: x * x
>>> f
<function <lambda> at 0x101c6ef28>
>>> f(5)
25
同样，也可以把匿名函数作为返回值返回，比如：def build(x, y):
    return lambda: x * x + y * y
Python对匿名函数的支持有限，只有一些简单的情况下可以使用匿名函数。
